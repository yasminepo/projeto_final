COMO SURGIU?

A ideia surgiu do exemplo de projeto apresentado pelo professor na aula sobre funções,
o qual realizava o cadastro de usuários. Com isso, foi pensado em um programa que
além de cadastrar usuários, também fosse útil na vida de muitas pessoas e utilizasse
todos os conteúdos aprendidos na disciplina, surgindo assim a ideia de um “gerenciador
de convidados”.

PARA QUE SERVE?

Um gerenciador de convidados serve para controlar a lista de convidados de um evento,
o que também pode ser feito em uma folha de papel, porém um programa oferece uma
forma mais rápida e organizada de administrar essas informações.

FUNCIONALIDADES

CADSTRAR CONVIDADO:

Possibilitará ao usuário adicionar um convidado para o evento.

EDITAR CADASTRO:

Possibilitará ao usuário fazer alterações nos dados de um convida já cadastrado.

CONFIRMAR PRESENÇA:

Possibilitará ao usuário registrar a confirmação de presença de um convidado.

CANCELAR PRESENÇA:

Possibilitará ao usuário cancelar a confirmação de presença de um convidado.

EXCLUIR CONVIDADO:

Possibilitará ao usuário remover um convidado da lista de convidados do evento.

LISTAR CONVIDADOS:

Possibilitará ao usuário visualizar todos os convidados já cadastrados.

PESQUISAR CONVIDADO:

Possibilitará ao usuário encontrar usuários pelo nome utilizado no cadastro.

MOSTRAR RESUMO:

Possibilitará ao usuário ver o número de convidados, confirmações e pendências.
